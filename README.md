## Usage

You **cannot** deploy Waves on any static hosting services!

## Self-Hosting
   
```bash
git clone https://github.com/l4uy/waves/

cd waves

sh setup.sh
```

## Development & Building

Development

```bash
git clone https://github.com/l4uy/waves/

cd waves

bun i

bun dev
```

Production

```bash
git clone https://github.com/l4uy/waves/

cd waves

bun i

bun run build

bun start
```
